# ElevenLabs Studio 3.0 Voiceover Pack

This folder contains the script and subtitle assets prepared for the current Sharp GUI intro v9 render.

## Files

- `sharp-gui-v9-voiceover-script.txt`  
  Clean narration text. Paste this into ElevenLabs Studio's voiceover script box.

- `sharp-gui-v9-video-script.md`  
  Timestamped production script with visual beats, narration windows, and delivery notes.

- `sharp-gui-v9-voiceover.zh-CN.srt`  
  Timed Chinese subtitles matching the narration script.

- `sharp-gui-v9-voiceover.zh-CN.vtt`  
  WebVTT version of the same narration subtitles.

- `sharp-gui-v9-visual-captions.zh-CN.srt`  
  Timed SRT for the existing on-screen captions already shown in the HyperFrames video.

## Recommended ElevenLabs Setup

1. Upload `renders/sharp-gui-intro-v9-hq-1080p.mp4`.
2. Add voiceover and paste `sharp-gui-v9-voiceover-script.txt`.
3. Keep each paragraph as its own narration paragraph/clip if Studio preserves paragraph breaks.
4. Use the timeline to align each paragraph to the timing windows in `sharp-gui-v9-video-script.md`.
5. Use `sharp-gui-v9-voiceover.zh-CN.srt` if you need external subtitles for the generated narration.

For a natural Chinese product-film tone, prefer a calm adult voice with low-to-medium energy. Avoid over-bright promo voices. If a technical term is pronounced poorly, try a pronunciation dictionary or rewrite the term phonetically in the script copy inside ElevenLabs, while keeping the subtitle file unchanged.
